* Installation
1. Add ~$HOME/.mix/escripts~ to your ~PATH~ variable
2. Install escript
#+BEGIN_SRC shell
mix escript.install hex bt
#+END_SRC

#+RESULTS:

* Usage
1. List devices
  #+BEGIN_EXAMPLE
  $ bt devices

  AirPods
  #+END_EXAMPLE

2. Add new alias
  #+BEGIN_EXAMPLE
  $ bt alias add

  1. AirPods
  Select device: 1
  Enter alias: a
  #+END_EXAMPLE

3. List your aliases
  #+BEGIN_EXAMPLE
  $ bt alias ls

  a -> AirPods
  #+END_EXAMPLE

4. Connect
  #+BEGIN_EXAMPLE
  $ bt con a

  Trying to connect... done
  #+END_EXAMPLE
